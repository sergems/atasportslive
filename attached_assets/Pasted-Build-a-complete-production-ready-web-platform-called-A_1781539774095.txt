Build a complete production-ready web platform called Advanced Talent Agency (ATA).
Project Overview
Advanced Talent Agency (ATA) is a sports and entertainment talent agency based in Kampala, Uganda.
The platform consists of:

1. ATA Sports Live Streaming Platform
Admins must be able to update the information on the home page and update them. there must be Upcoming Streams details on the home page, which must display the upcoming streams and details about that stream, there must be a countdown to that stream.
Users can:

Register and login
Deposit funds into their wallet
Watch premium live-streamed Pool matches
Access exclusive sports content

Live streaming access is subscription-based:

Cost: USD $1.50 per day
A user can only watch live matches if they have sufficient wallet balance.
The system automatically deducts $1.50 every 24 hours while the user continues watching premium content.
If the balance becomes insufficient, access is immediately suspended.
Users receive low balance notifications.
2. Peer-to-Peer Betting Exchange

ATA does NOT act as a bookmaker.

ATA acts only as a broker/matching exchange.

Users bet against each other.

Supported sports:

Pool
Boxing

Example:

User A:

Match: Pool Match #100
Outcome: Player A wins
Stake: $100

User B:

Match: Pool Match #100
Outcome: Player B wins
Stake: $100

The system automatically matches both bets.

ATA deducts a 10% brokerage fee from the total pool.

The winner receives the remaining amount.
